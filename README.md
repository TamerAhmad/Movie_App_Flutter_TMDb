# Title

Movie application with TMDb API to display popular movies and top movies. With cool design.
Was applied by Flutter and API.

## Overview

The App Has:
- Popular Movies Interface
- Top Movies Interface
- Loading Interface

Movie attributes: (poster, backposter, title, overview, year, rate, language, family check)

**Note: you need to provide TMDb api key for application work well with you. Located at (lib/tools/api_key.dart)**
